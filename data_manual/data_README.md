# Data Directory for Manually-Created Data stored on Local Repository

This folder contains datasets used in the replication and extension of **Jiang et al. (2024)** on bank balance sheet exposure to interest rate risk.

The data include regulatory bank filings, market price data, and intermediate datasets generated by the analysis pipeline.

---

# 1. Regulatory Bank Data

## FFIEC Call Reports

Files:
- `FFIEC CDR Call Bulk All Schedules 03312022.zip`
- `FFIEC CDR Call Bulk All Schedules 12312025.zip`

Quarterly **Call Report bulk data** downloaded from the FFIEC Central Data Repository.

Source:  
https://cdr.ffiec.gov/public/PWS/DownloadBulkData.aspx

These files contain balance sheet and income statement information for all U.S. commercial banks.

They are used to construct the processed bank panel datasets.

---

## Processed Bank Panels

Files:
- `bank_panel_03312022.parquet`
- `bank_panel_12312025.parquet`

These datasets are created from the raw Call Reports and contain bank-level balance sheet information including:

- Total assets  
- Uninsured deposits  
- Treasury securities  
- Mortgage-backed securities  
- Loan portfolios by category  
- Asset maturity buckets  

These files form the **main analysis dataset** used in the replication.

---

# 2. Market Data

## Treasury Yields

File:
- `treasury_yields.parquet`

Contains U.S. Treasury yields for multiple maturities:

- 1-year  
- 3-year  
- 5-year  
- 10-year  
- 20-year  
- 30-year  

Source: Federal Reserve Economic Data (FRED)

Used to compute interest rate shocks.

---

## Mortgage-Backed Securities Prices

File:
- `mbs_etfs.parquet`

Contains price data for mortgage-backed securities ETFs used as proxies for MBS price movements.

Used to estimate market price changes in bank MBS portfolios.

---

## Macro Data (FRED)

Files:
- `fred.csv`
- `fred.parquet`

Contains macroeconomic and financial time series pulled from the **FRED database**, including:

- GDP  
- CPI  
- Federal Funds Rate  
- Federal Reserve balance sheet variables  

---

# 3. Derived Data

## Market Shocks

File:
- `market_shocks.parquet`

Contains calculated interest rate shocks used in the analysis:

- Treasury price shocks by maturity  
- MBS price changes  
- shock multipliers used for mark-to-market loss calculations  

Generated by: src/compute_market_shocks.py


---

## GSIB Bank Classification

File:
- `gsib_list.parquet`

Contains the RSSD identifiers for **Global Systemically Important Banks (GSIBs)**.

Used to classify banks into:

- Small banks  
- Large non-GSIB banks  
- GSIBs  

This classification is used in the replication of **Table 1**.

---

# 4. Repo Market Data

Files:

- `repo_public.parquet`
- `repo_public_relative_fed.parquet`
- `repo_public_relative_fed.xlsx`
- `ofr_public_repo_data.parquet`

These datasets contain public repo market data used for exploratory analysis of short-term funding conditions.

Source: Office of Financial Research (OFR)

---

# Reproducibility

Most datasets in this folder are **automatically generated by scripts in** src/ 

The only external data required to fully reproduce the pipeline are the raw **FFIEC Call Report bulk files**, which can be downloaded from the FFIEC website.
